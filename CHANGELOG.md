# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - YYYY-MM-DD

### Added

- macOS menubar utility for display color adjustments
- Overlay-based dimming and color tinting
- Display transfer table manipulation for brightness, contrast, gamma and temperature
- Preset management: create, edit, duplicate, delete and reorder
- Persistent storage of presets and session state
- Automatic display restoration on quit and crash recovery
- Structured logging via `os.Logger`
